Hello-World-Serveranwendung
Auf jeden Request soll immer die gleiche HTML5-Seite ausgeliefert werden.

package.json erstellen:
In der Shell in den Ordner navigieren und npm init eingeben
Dann die Zeilen ausfüllen.

express installieren:
npm install --save express
Dadurch wird express in der package.json als dependencie hinzugefügt.

app.js aufbauen:
use strict um JS einzuschränken
Requires um Module verwenden zu können
Methode um alle request abzufangen und passende Antwort anbieten
Funktion um Server zu starten
